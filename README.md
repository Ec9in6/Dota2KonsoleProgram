# Dota 2 Stats (Kotlin Multiplatform)
Десктопное приложения которое помогает отлеживать статистику в Dota 2 

### Стек 
* Kotlin
* UI (Kotlin Multiplatform)
* Ktor Client
* Kotlinx Serialization

### Структура проекта
Я разделил код на логические части, чтобы в нем было проще ориентироваться:
* **src/main/kotlin/model/** — здесь лежат описания данных (модели), которые мы получаем от API (имя игрока, его аватарка, количество побед и т.д.).
* **src/main/kotlin/api/** — здесь находится сетевой клиент, который делает запросы к серверу OpenDota.
* **src/main/kotlin/Main.kt** — главный файл, в котором написан весь интерфейс приложения (окно, кнопки, текстовые поля и логика отображения).
